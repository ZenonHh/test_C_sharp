   > run-admin.bat

   ✓ MAUI app: Xem quán ăn, xem bản đồ, nghe audio
   ✓ Web Admin: Quản lý dữ liệu (CRUD)

❓ DÙNG VỚI XAMPP:

Nếu cần chạy trên XAMPP:

1. Set environment:
   $env:VINHKHANH_DB_PATH = "C:\xampp\htdocs"

2. Copy database vào XAMPP:
   Copy "...AppData\Roaming\VinhKhanhTour\VinhKhanhTour_Full.db3" 
        "C:\xampp\htdocs\"

3. Chạy trên port 80:
   dotnet run --urls "http://localhost:80"
