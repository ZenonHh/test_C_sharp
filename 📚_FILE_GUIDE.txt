🎓 ĐỂ THUYẾT TRÌNH:

"Tôi đã phát triển một hệ thống MAUI + Web Admin kết nối qua API.

MAUI Mobile App gọi Web API để lấy dữ liệu,
Web Admin quản lý SQLite Database và cung cấp API.
2 bên share cùng database → tự động sync.

Nếu Web Admin down, MAUI vẫn dùng local DB fallback.
Architecture kinh điển Client-Server,
dễ scale, dễ maintain, dễ extend thêm clients khác."
