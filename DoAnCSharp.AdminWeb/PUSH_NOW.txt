## 🔴 If It Fails

### Connection Error?
```powershell
git config --list | grep remote
```

### Authentication Error?
```powershell
# Setup credentials
git config --global user.email "your-email@gmail.com"
git config --global user.name "Your Name"
```

### Then retry push:
```powershell
git push origin main
```
